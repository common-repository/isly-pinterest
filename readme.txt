Pinterest Pin-It buttons that reference post permalinks.

== Description ==

Gives your in-post images little Pinterest rollover buttons that allow users to tag your posts' permalinks instead of the main page, search page, archives or wherever else they found the image.  Someone should have done this a long time ago.

== Installation ==

1. Upload `plugin-name.php` to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
2. Tweak your settings via Settings -> Isly Pinterest
